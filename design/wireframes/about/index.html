License & Attribution

SeriesScraper — Private web application for scraping and matching forum content against IMDB datasets.

IMDB Datasets: Non-commercial use only. Data provided by IMDb.com.

For more information about IMDB's dataset licensing, visit IMDb Non-Commercial Datasets.

This application is intended for private, non-commercial use only.

Technology Stack

  • Framework: .NET 8 with Blazor Server
  • Database: PostgreSQL 16
  • ORM: Entity Framework Core
  • UI Components: MudBlazor
  • Testing: xUnit
  • Containerization: Docker + Docker Compose
  • CI/CD: GitHub Actions (self-hosted runner)
